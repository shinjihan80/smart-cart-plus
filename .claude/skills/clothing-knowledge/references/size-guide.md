# 의류 사이즈·소재 상세 가이드

## 소재별 특성 및 두께 분류

### 얇음 (여름용 / 통기성 중심)
| 소재 | 특성 | 대표 제품 |
|------|------|---------|
| 리넨 | 통기성 최상, 흡습성 좋음 | 여름 셔츠, 원피스 |
| 시어 / 쉬폰 | 반투명, 가벼움 | 블라우스, 스카프 |
| 레이온 (가벼운) | 부드럽고 드레이프감 | 여름 원피스, 롱스커트 |
| 모달 | 부드럽고 흡습 | 속옷, 티셔츠 |

### 보통 (사계절용)
| 소재 | 특성 | 대표 제품 |
|------|------|---------|
| 면 (Cotton) | 범용, 세탁 용이 | 티셔츠, 청바지 |
| 폴리에스터 | 구김 적고 관리 편 | 슬랙스, 재킷 |
| 데님 | 내구성 강함 | 청바지, 데님 재킷 |
| 스판덱스 혼방 | 신축성 좋음 | 레깅스, 요가복 |

### 두꺼움 (겨울용 / 보온 중심)
| 소재 | 특성 | 대표 제품 |
|------|------|---------|
| 울 (Wool) | 보온성 최상, 고급감 | 코트, 니트 |
| 기모 | 안감이 따뜻함 | 기모 후리스, 기모 바지 |
| 플리스 | 가볍고 따뜻 | 플리스 재킷 |
| 패딩 / 다운 | 보온성 극상 | 패딩 점퍼, 다운 코트 |
| 캐시미어 | 고급 보온 | 니트, 머플러 |
| 가죽 / 합성피혁 | 방풍 효과 | 가죽 재킷 |

## 카테고리 분류 기준

### "의류"로 분류
- 상의: 티셔츠, 셔츠, 블라우스, 니트, 후드티, 맨투맨
- 하의: 바지, 스커트, 레깅스, 반바지
- 아우터: 코트, 재킷, 패딩, 점퍼, 가디건
- 원피스 / 세트
- 신발, 양말

### "액세서리"로 분류
- 주얼리: 반지, 목걸이, 귀걸이, 팔찌
- 가방: 숄더백, 토트백, 크로스백, 백팩
- 모자, 비니, 벨트, 스카프, 머플러
- 시계
