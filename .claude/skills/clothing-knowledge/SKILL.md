# Clothing Knowledge — 의류 분류 도메인 지식

의류·액세서리의 소재(`material`), 두께(`thickness`), 사이즈(`size`), 날씨 태그(`weatherTags`) 분류 기준을 제공한다.

## 사용 시점

receipt-parser가 다음 상황에서 참조:
1. 텍스트에 소재·두께가 명시되지 않은 경우
2. 상품명에서 속성을 추론해야 하는 경우

## 소재별 두께 매핑

| 소재 키워드 | thickness | 특성 |
|------------|-----------|------|
| 리넨, 시어, 쉬폰 | 얇음 | 여름/통기성 |
| 면, 폴리에스터, 레이온 | 보통 | 사계절 |
| 울, 니트, 플리스, 기모, 패딩, 다운 | 두꺼움 | 겨울/보온 |
| 데님, 청바지 | 보통 | 사계절 |
| 가죽, 합성피혁 | 두꺼움 | 가을/겨울 |

## 두께별 날씨 태그 기본 매핑

| thickness | 기본 weatherTags |
|-----------|-----------------|
| 얇음 | ["여름", "맑음"] |
| 보통 | ["봄", "가을"] |
| 두꺼움 | ["겨울"] |

## 카테고리 분류 기준

| 유형 | category |
|------|----------|
| 상의, 하의, 원피스, 아우터, 신발 | "의류" |
| 반지, 목걸이, 귀걸이, 팔찌, 가방, 모자, 시계 | "액세서리" |

## 브랜드별 기본 사이즈 체계

| 브랜드 패턴 | 사이즈 체계 |
|------------|-----------|
| 유니클로, 자라, H&M | S/M/L/XL |
| 나이키, 아디다스 | 한국 사이즈 (240–280) |
| 팬도라, 티파니 | 수치 (반지: 45–60, 팔찌: Free) |

## 불확실 시 기본값

- `thickness: "보통"`, `material: "혼방"`, `size: "M"`, `weatherTags: ["봄", "가을"]`
