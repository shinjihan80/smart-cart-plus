# Food Knowledge — 식품 보관 도메인 지식

식품 카테고리별 권장 보관 방법(`storageType`)과 기본 보관 기간(`baseShelfLifeDays`)을 제공한다.

## 사용 시점

receipt-parser가 다음 상황에서 참조:
1. 텍스트에 보관 방법이 명시되지 않은 경우
2. `baseShelfLifeDays`를 추정해야 하는 경우

## 빠른 참조표

| 카테고리 | storageType | baseShelfLifeDays |
|----------|-------------|-------------------|
| 신선 채소/과일 | 냉장 | 3–7 |
| 유제품 (우유, 요거트) | 냉장 | 7–14 |
| 두부·콩나물 | 냉장 | 3–5 |
| 육류 (냉장) | 냉장 | 2–3 |
| 육류 (냉동) | 냉동 | 30–90 |
| 생선·해산물 (신선) | 냉장 | 1–2 |
| 생선·해산물 (냉동) | 냉동 | 30–60 |
| 계란 | 냉장 | 21–28 |
| 가공식품 (상온) | 실온 | 90–365 |
| 라면·과자 | 실온 | 180–365 |
| 쌀·곡물 | 실온 | 365 |

## 불확실 시 보수적 기본값

카테고리 판단 불가 시: `storageType: "냉장"`, `baseShelfLifeDays: 3`
