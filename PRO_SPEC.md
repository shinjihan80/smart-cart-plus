## ⭐ Pro 핵심 기능

### 1. 무제한 AI 호출

- 베이직의 4개 에이전트 한도 전부 제거
- `aiQuota.ts`의 `remaining()`이 Pro 사용자에 대해 `Infinity` 반환하도록 확장
- 사용량 대시보드는 계속 표시 (정보성)
- **개발 노트**: 서버측 rate-limiting 별도 (비정상 사용 방지)

### 2. 클라우드 자동 동기화

- **Supabase** 기반 — 인증 + Row Level Security
- 주요 도메인 키 서버 미러:
  - `smart-cart-items`, `smart-cart-archive`, `smart-cart-history`
  - `nemoa-wear-log`, `nemoa-cook-log`
  - `nemoa-profiles`, `nemoa-shopping-list`
  - `nemoa-saved-outfits`, `nemoa-recipe-favorites`
- **동기화 전략**: Last-Write-Wins + 충돌 시 alert → "이 기기 우선 or 서버 우선" 선택
- **오프라인 지원**: 로컬 우선 쓰기, 온라인 복귀 시 큐 flush
- **30일 버전 이력** — 실수 삭제 복원 가능

### 3. 파트너 연동 활성화 (Phase C 이후)

현재 베이직에서 `partnerLinks.ts`에 **9개 파트너 스텁** 등록돼 있고 전부 `enabled: false`. Pro 출시 시 활성화.

| 도메인 | 파트너 | 수수료 모델 |
|--------|--------|------------|
| 식품 쇼핑 | **쿠팡파트너스** | 3-5% (딥링크) |
| 식품 쇼핑 | **마켓컬리 B2B** | 건당 수수료 (계약 예정) |
| 식품 쇼핑 | **네이버 쇼핑** | CPC/CPS |
| 대형마트 | **홈플러스 · 이마트** | CPA (스텁) |
| 중고 의류 | **당근마켓** | 건당 수수료 (계약 예정) |
| 기부 | **아름다운가게** | 비영리 (노출만) |
| 세탁/수선 | **세탁특공대** | 건당 수수료 |
| 짐 보관 | **미니창고 다락** | 첫 결제 수수료 |

**구현**:
- `partnerLinks.ts`에서 Pro 사용자 한정으로 `enabled: true` 토글
- `PartnerChip` 컴포넌트는 이미 enabled/disabled 분기 로직 있음 → 로직 수정 최소

### 4. 프리미엄 레시피 100종+ 추가

- 현재 베이직 42종 → Pro에서 **142종+**
- 계절별 40종 추가 (봄/여름/가을/겨울 각 10종)
- 테마별 세트:
  - "1인 가구 5분 레시피" 20종
  - "아이 이유식·유아식" 15종
  - "비건·채식" 15종
  - "명절 한식" 10종 (추석·설)

### 5. 저장 코디 · 프로필 무제한

- `savedOutfits.ts` 20개 제한 해제
- `profile.ts` 2명 제한 해제
- UI에서만 제한 걸려 있으므로 Pro 플래그로 분기 1줄

### 6. 고급 지출 분석

- 연간 리포트 (카테고리별 · 월별 히트맵)
- 카테고리 예산 설정 + 알림 ("이번 달 의류 예산 80% 사용")
- 가족 프로필별 지출 분담 분석 (Pro+ 검토)

### 7. 우선 고객 지원

- 이메일 24시간 내 답변 보장
- 문의 시 `errorLog` 자동 첨부 옵션
- 대형 버그 발견 시 무료 연장 1개월

## 베이직 → Pro 전환 플로우

1. 베이직 사용자가 한도 도달 시 (AI 한도 소진, 저장 코디 21개째 등) Pro 모달
2. "Pro 1개월 무료 체험" — 결제 수단 등록 전 체험 허용 (전환율 상승 목적)
3. 체험 종료 3일 전 알림
4. 체험 → 유료 전환 시 토스페이먼츠 빌링 등록
5. 해지는 설정에서 1클릭 (다음 갱신일까지 Pro 유지)

## Pro로 **안 옮기는** 기능 (원칙)

베이직이 "체험판"으로 느껴지지 않도록 다음은 절대 Pro 전용으로 돌리지 않음:

- ❌ 로컬 저장 자체 (베이직에서도 영구)
- ❌ 백업 JSON 다운로드 (수동 복원 항상 무료)
- ❌ 레시피 검색·필터 (Pro는 개수만 확장)
- ❌ 날씨·제철 정보 (핵심 가치)
- ❌ 기본 AI 기능 자체 (한도만 다름)
- ❌ 기본 42종 레시피 (축소 없음)
- ❌ 착용/조리 로그 분석 (개인화 핵심)

**원칙**: 베이직은 "혼자 쓰기 충분한 완성품", Pro는 "멀티 기기·무제한·제휴"의 업그레이드.
