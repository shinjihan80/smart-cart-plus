# NEMOA v1.3

**일상을 반듯하게 모으다** — 스마트 냉장고와 옷장을 하나로, AI 라이프스타일 비서 네모아

> 사각형 벤토 UI(네모)에 사용자의 식(食)과 의(衣) 데이터를 자동 수집(모아)하고,
> 네모아가 보관 기한·코디·재구매까지 챙기는 모바일 퍼스트 앱.

---

## 주요 기능

### 홈 대시보드 (9개 위젯)
- NEMOA 브랜드 헤더 + 시간대별 네모아 인사
- 데일리 브리핑 (7시간대별 날씨 + 코디 추천)
- 냉장고 카루셀 (D-Day + 이미지 + 스와이프 소진)
- 월별 소비 내역 (바 차트 + 탭 전환)
- 네모아의 주간 인사이트 (보유 데이터 기반 맞춤 팁)
- 전체 상품 검색 (썸네일 + 실시간 필터링)

### 스마트 냉장고
- 식품 11종 카테고리 (채소, 정육, 수산, 유제품 등)
- D-Day 프로그레스 바 + 영양 정보 + 이미지
- 2단 필터 (보관 타입 + 식품 그룹) + 정렬
- 네모아가 추천하는 오늘의 메뉴 (8종 레시피, 보유 식재료 매칭)
- 재구매 추천 (소진 히스토리 기반)
- 빠른 추가 (이미지 포함 원탭 등록)

### 스마트 옷장
- 패션 13종 카테고리 (상의, 하의, 신발, 가방, 액세서리 등)
- 계절 추천 + 네모아가 추천하는 오늘의 코디
- 가상 코디 미리보기 (이미지 슬롯)
- 빠른 추가 (이미지 포함 원탭 등록)

### 마이페이지
- 종합 통계 + 카테고리 분포 차트
- 월별 지출 추이 (spring 바 차트)
- 알림 토글 3종 + 데이터 내보내기 (JSON/CSV)
- 쇼핑몰 연동 설정 (준비 중)

### 관리자 대시보드 (`/admin`)
- 에이전트 6종 헬스체크 + 실시간 Live 모드
- 토큰 사용량 추정 + 캐시 히트율
- 데이터 품질·스키마·localStorage 모니터링
- 시스템 제어 5종 (아카이브/재검증/로그 비우기/디버그 스냅샷/리셋)

### 상품 등록 (4가지 방법)
| 방법 | 이미지 | 설명 |
|------|--------|------|
| 사진 분석 | 자동 첨부 | 네모아 Vision이 식품/패션 자동 분류 |
| URL 분석 | og:image 자동 | 쇼핑몰 페이지에서 상품 정보 추출 |
| 빠른 추가 | Unsplash 자동 | 자주 구매하는 아이템 원탭 등록 |
| 텍스트 입력 | 수동 | 영수증/이메일 텍스트 네모아 파싱 |

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 16.2.4 (App Router) |
| 언어 | TypeScript (strict) |
| 스타일 | Tailwind CSS v4 + SUIT 폰트 |
| 애니메이션 | Framer Motion (spring) |
| 아이콘 | Lucide React |
| AI | Anthropic Claude API (Opus 4.6 / Haiku 4.5) |
| 상태 관리 | React Context + localStorage |
| PWA | manifest.json + standalone |

## 시작하기

```bash
npm install
npm run dev
# http://localhost:3000
```

## 프로젝트 구조

```
src/
├── app/           # 5 페이지 (홈/냉장고/옷장/마이/관리자) + 8 API 라우트
├── components/    # UI 컴포넌트 + NemoaLogo
├── context/       # CartContext + ToastContext
├── types/         # 24종 카테고리 타입 시스템
├── lib/           # AI 파이프라인 + 관리자 통계 + 에이전트 로거
└── data/          # Mock 데이터 (22개 아이템)
```

- **폰트:** SUIT (워드마크는 `font-extrabold tracking-tight`)
